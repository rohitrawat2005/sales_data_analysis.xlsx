# Sales Data Analysis (Google Sheets)

## Overview
This project analyzes a global sales dataset using Google Sheets. The goal was to clean transactional data, handle missing values appropriately, and extract business insights using pivot tables and charts.

## Tools Used
- Google Sheets
- Pivot Tables
- Charts (Bar & Line)

## Key Analysis
- Country-wise sales distribution
- Monthly sales trend analysis
